RCS Serial Thermostats Plugin for HomeSeer

Introduction

Thank you for the purchase of the HomeSeer RCS Serial Thermostats Plugin!  The HomeSeer RCS Serial Thermostats plugin gives you total automation control of your RCS Serial Thermostats. 

Requirements

There are several requirements to using the HomeSeer RCS Serial Thermostats plugin:
  • HomeSeer Version 3.0.0.0 or higher must be installed.
  • An RCS Serial Communicating thermostat or Zone Controller must be installed.  The plugin supports the TR16, TR40, and ZC6R model Thermostats
  • The Thermostat or Communications module must be connected to your HomeSeer computer via a COM port.

Config Page 

Information

This screen is used to control the settings for communicating with the thermostats.  It can be accessed by selecting "plugins", "RCS Stat", then "RCSSerial Config". 

Page Items

 Poll Interval

This setting determines the how often HomeSeer will retrieve the status from the panel, in seconds. 

Enable Debug Messages

This box will enable certain log messages that may help to debug any problems with the plugin.

Baud Rate

This sets the baud rate to allow communication with the physical device at a speed the physical device can handle. (default is 9600)

Save Settings

This button will save the settings listed above to the RCS Serial Configuration file. 

Add Thermostat

Enter a thermostat name in the Box above and click this button to add it to the system

Enter a primary location in the box of the same name. This corresponds to the primary location value in the device management page.

Enter a secondary location in the box of the same name. This corresponds to the secondary location value in the device management page.

Enter the address of the thermostat. This corresponds to the setting configured on the SW1 dipswitch of the thermostat .

Thermostat List

This lists all thermostats configured in the system.  Click a thermostat, then click "Edit Thermostat" to change its settings

Edit Thermostat

Selecting a thermostat from the droplist will display the following configuration properties:


Name

This value must be changed on the device management page.

Location

This value must be changed on the device management page.

Location 2

This value must be changed on the device management page.

Address

This is the address of the thermostat as configured in the RCS Serial system.  These usually range from 1 to 15.  See your RCS Serial documentation for more information on how to change the address of the thermostat. 

 

Basic Control Instructions For Device Management Page

To Change Setpoints

  1. Enter the value of the heat or cool setpoint in the appropriate box

To Change fan mode

  1. Click the "Fan On" or "Fan Auto" button on the page.

To Change the Operating Mode

  1. Click on the "Heat", "Cool", "Auto", "Aux" or "Off" button to change the mode

To Change The Hold Mode

  1. Click the "Hold" or "Run" button on the page.
